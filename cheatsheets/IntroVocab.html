Introduction to Computer Science Vocabulary

word definition example context
number system base the factor by which the place values in a number system increase by (all number systems start at 1) decimal, binary, octal, hexadecimal In decimal, the base is 10 so place values are 1, 10, 100, 1000, etc.; In binary, the base is 2 so place values are 1, 2, 4, 8, etc.
place value a spot in a number system to hold a place In the number 435: the 4 is in the 100's place, 3 is in the ten's place and 5 is in the one's place Even if there's nothing in the place value, you need to put a zero to "hold the place"
decimal base 10: the number system that humans use. Digits go from 0 to 9 1234: one-thousand, two-hundred, thirty, four This is what humans have always used (perhaps because we have 10 fingers). We assume that any number is in base 10 unless otherwise stated but we can make it explicit with the "0d" prefix.
binary base 2: the number system that computers use: digits go from 0 to 1 0b1011: one-eight, no-four, one-two, one-one: equivalent to the 0d11 Computers use this because it makes the electronics easier to build. We know it's binary because of the "0b" prefix.
hexadecimal base 16: the number system that humans use to read binary data. Digits go from 0 to F (15) 0xA90C: ten-4096's, nine-256's, no-16's, twelve-one's Humans use this because it's easy to translate from binary and it's a lot more compact
ASCII American Standard Code for Information Interchange 65 is the capital "A" The order doesn't matter as long as everyone knows and uses the STANDARD.
lumens a measure of the brightness of a light A standard flashlight puts out about 300 lumens Multiple light sources cna be combined to provide incrementally more light
hue
saturation
Color channel Colors are physically created by a combination of 3 primary colors each or which represents one channel The colors used for computer screens are red, green, and blue
BPC Bits Per Color Most modern JPG images use a 24-bit color which makes for 8-bits per channel (8 for red, 8 for green, 8 for blue) If the number of bits per pixel cannot be divided evenly by three, the color is typically represented as a greyscale image instead.
variable a name which represents a value the variable "age" has a value of 8 The value of a variable can "vary" over time. Once they are assigned a new value, the old value is lost.
function a process which takes a value as input and returns a value as output This is a common thing in mathematics: f(x) = x^2 + 3x + 4: here the input is some number "x" and the output is the value that the equation produces when x is replaced by that value In mathematics, a function is a little more strictly defined. In computer science, a function need not have an input and may not produce an output.
syntax the rules that must be followed to produce valid code every open parenthesis must be matched by a close parenthesis If a code does not follow the correct syntax, a syntax error will be produced when you try to run the code. The error is often quite helpful for identifying what is wrong but can sometimes be misleading.
