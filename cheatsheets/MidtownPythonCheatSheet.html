Python Cheatsheet

The Python syntax found here will be used continually throughout this course. It should be committed to memory. If you have to constantly be referring back to it, the assignments will take you twice as long to do and you will lose precious time. The purple boxes are the pattern (syntax) that you will always follow, the grey boxes show a specific example of actual code that follows that pattern.

Call a function
function_name(parameter_value_1, parameter_value_2) print("hello") input("Name: ")
Note: not all functions take parameters but even in these cases you still have to include the parenthesis (just don't put anything inside)
Create a variable or give an existing one a new value
variable_name = initial_value pi = 3.14
Note: variable must be on the left side
Naming rules
  1. Cannot be a reserved keyword
  2. Cannot start with a number
  3. No punctuation, delimeters, or special characters
  4. No spaces
Recommended to use snake case
Use Snake Case
put_underscore_between_words
Use when appropriate: when the purpose of the variable can't be summarized in a single word

A note on operators: generally the operators only work on values of the same type and the result will also be of that type. If you try, for example, to add something to a str[ing] that isn't itself another str[ing] you will get an error.
Import something from a package (AKA module)
Note: all imports should always be the first thing in your program (at the top)
from package_name import thing from random import randint
Note: this assumes the package is already installed on your machine. Python comes with a bunch of packages that are installed by default.
Comment out a line of code
# line of code you don't want to run anymore # print(red)
Comment out multiple lines of code
''' Multiple lines of code that you don't want to run anymore ''' ''' print(red) print(green) print(blue) '''
Execute code a specific number of times
for numberName in range(parameters): # Code to run for each number in the sequence # Prints out the numbers from 0 to 99 for num in range(100): print(num)
With 1 parameter, range(num1) produces a sequence of numbers from 0 up to but not including num1
With 2 parameters, range(num1, num2) produces a sequence of numbers from num1 up to but not including num2
With 3 parameters, range(num1, num2, num3) produces a sequence of numbers from num1 up to but not including num2 going up (or down if negative) by num3
